# 🌐 Personal Portfolio Landing Page UI

Uma interface moderna de portfólio pessoal desenvolvida com foco em apresentação profissional, identidade visual e experiência do usuário. O projeto apresenta uma landing page elegante para desenvolvedores e designers exibirem suas habilidades, serviços e informações de contato de forma atrativa.

---

## 📸 Preview do Projeto

Interface contendo:

- Logo personalizada
- Barra de navegação moderna
- Seção Hero de apresentação
- Título de destaque
- Descrição profissional
- Botão Call-To-Action (CTA)
- Links de navegação
- Layout moderno e responsivo

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Interface moderna e profissional
- Tema escuro elegante
- Gradiente visual sofisticado
- Tipografia impactante
- Layout minimalista
- Botões estilizados
- Navegação intuitiva
- Design totalmente responsivo
- Excelente hierarquia visual

---

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de uma landing page de portfólio pessoal, explorando conceitos de layout moderno, identidade visual, responsividade e componentes reutilizáveis com HTML e CSS.

---

## 📚 Funcionalidades

- Apresentação profissional do desenvolvedor
- Menu de navegação funcional
- Seção de introdução (Hero Section)
- Botão de chamada para ação
- Estrutura preparada para integração com projetos reais
- Layout adaptável para diferentes dispositivos
- Fácil personalização de conteúdos e estilos

---

## 📂 Seções da Página

- Home
- About
- Services
- Contact
- Login
- Register
